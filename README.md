# RemoveCharAtIndex
Ekrandan bir string bir de sayý alan (aralarýnda virgül ile), ilgili string ifade içerisinden verilen indexteki karakteri çýkartýp ekrana yazdýran console uygulasýný yazýnýz.

Örnek: Input: Algoritma,3 Algoritma,5 Algoritma,22 Algoritma,0

Output: Algritma Algortma Algoritma lgoritma
